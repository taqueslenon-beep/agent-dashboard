Pilha de Protocolos Agênticos

Taques Agents implementa os 3 pilares do ecossistema agêntico moderno, unificando a comunicação entre agentes, ferramentas e usuários.

MCP · Agente ↔ Ferramentas A2A · Agente ↔ Agente AG-UI · Agente ↔ Usuário

Eventos AG-UI Suportados

16 tipos de eventos padrão: ciclo de vida, transmissão de texto, chamadas de ferramenta, instantâneos/deltas de estado (RFC 6902), passos de raciocínio, rastreamento de atividade, interrupções (HITL), eventos personalizados.

Transporte

Agnóstico de protocolo. Suporta SSE, WebSockets e binário HTTP. Estado sincronizado via JSON Patch (RFC 6902). SDKs: TypeScript, Python, Go.
